# Web Scaffold

This is a minimal Next.js + Ant Design scaffold for the RAG-Py control panel.

## Run

```bash
npm install
npm run dev
```

## Planned feature folders

- `src/features/knowledge-bases`
- `src/features/documents`
- `src/features/queries`

## Contract

- API base URL: `NEXT_PUBLIC_API_BASE_URL`
- Keep API access in `src/services/*`
- Keep feature logic inside `src/features/*`
